# Полный план тестирования PHP ↔ Node.js

Цель: проверить ВСЕ эндпоинты, ВСЕ query/body параметры, все граничные случаи.
Каждый тест — один и тот же запрос на PHP (:8082) и Node (:8081), сравнение ответов.

## 1. AUTH & SESSION

### 1.1 POST /:db/auth — логин
| # | Параметры | Описание |
|---|-----------|----------|
| 1 | `login=X&pwd=X` | Успешный логин, Set-Cookie |
| 2 | `login=X&pwd=X&JSON=1` | JSON ответ `{status,token}` |

| 4 | `login=X&pwd=wrong&JSON=1` | Неверный пароль, JSON |
